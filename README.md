# 🏥 Pronavakr-Apotek

A professional web-based management platform designed for pharmaceutical and retail inventory tracking. This project is optimized for high performance and hosted via GitHub Pages for seamless accessibility.

## 🌟 Key Features
* **Dynamic Data Rendering:** Uses centralized JavaScript data structures (`product_data.js` and `otc_data.js`) for easy updates.
* **Inventory Management:** Specifically structured to handle pharmaceutical and OTC (Over-The-Counter) product listings.
* **Clean UI/UX:** A minimalist and responsive interface designed for business efficiency.
* **Custom Domain Support:** Pre-configured with CNAME for professional branding.

## 📂 Project Architecture
| File | Description |
| :--- | :--- |
| `index.html` | The main entry point and UI layout of the application. |
| `product_data.js` | Contains the primary database for pharmaceutical products. |
| `otc_data.js` | Dedicated data file for non-prescription/OTC products. |
| `logo.png` | Official brand identity assets. |
| `CNAME` | Configuration for custom domain mapping. |
